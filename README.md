# 김민수 | 자기소개 페이지

소프트웨어 개발자 **김민수**의 자기소개 포트폴리오 웹사이트입니다.  
[brittanychiang.com](https://brittanychiang.com) 스타일을 참고한 다크 테마의 싱글 페이지로 제작되었습니다.

## 미리보기

브라우저에서 `intro.html` 파일을 열면 바로 확인할 수 있습니다.

## 주요 기능

- 반응형 레이아웃 (데스크톱 / 모바일)
- 사이드바 메뉴 호버 시 섹션 이동
- 경험 탭 호버 전환 (스타트업 · 해커톤 · 학회)
- 스크롤 등장 애니메이션
- 프로필 사진 표시

## 페이지 구성

| 섹션 | 내용 |
|------|------|
| 01. 소개 | 지원 동기 및 포부 |
| 02. 경험 | 스타트업, 해커톤, 학회 활동 |
| 03. 성격 | 장단점 |
| 04. 생활신조 | "기술은 사람을 위해 존재한다" |
| 05. 포부 | 단기 · 중장기 · 궁극적 목표 |

## 파일 구조

```
연습/
├── intro.html          # 메인 HTML
├── style.css           # 스타일
├── script.js           # 자바스크립트
├── P.png               # 프로필 사진
├── 자기소개서 실습용.txt  # 원본 자기소개서 텍스트
└── README.md           # 프로젝트 설명
```

## 사용 기술

- HTML5
- CSS3
- JavaScript (Vanilla)

## 로컬에서 실행하기

1. 이 폴더를 다운로드하거나 클론합니다.
2. `intro.html`을 더블클릭하거나 브라우저로 드래그합니다.

> `P.png`는 `intro.html`과 **같은 폴더**에 있어야 프로필 사진이 표시됩니다.

## GitHub Pages로 배포하기

1. GitHub에 새 저장소를 만듭니다.
2. 아래 파일을 저장소 루트에 업로드합니다.
   - `intro.html` → **`index.html`** 로 이름 변경 권장
   - `style.css`
   - `script.js`
   - `P.png`
3. **Settings → Pages → Source** 에서 `main` 브랜치, `/ (root)` 선택
4. 배포 후 `https://아이디.github.io/저장소명/` 주소로 접속
